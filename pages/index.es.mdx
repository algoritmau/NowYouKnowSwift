# Bienvenido a Learn iOS Dev

**Learn iOS Dev** es un proyecto de código abierto que busca ayudar a quienes quieran iniciar una carrera como desarrollador(a) de iOS. Este proyecto lo inicié documentando mi propia ruta de aprendizaje y espero poder mejorarlo y mantenerlo con la ayuda de la comunidad de desarrollo iOS para uso futuro.

## ¿Por qué este proyecto?

Este proyecto nació como una forma de documentación mientras aprendía desarollo iOS. Tomé múltiples cursos y consulté diversas referencias en línea para tratar de tener perspectivas e información diferentes sobre los temas. Más tarde, decidí expandir el proyecto y convertirlo en una especie de guía para ayudar a principiantes a ingresar al mundo del desarrollo iOS. 

## ¿Qué puedo aprender con este proyecto?

**Learn iOS Dev** incluye lecciones sobre [fundamentos de Swift](/swift-fundamentals/basics) —el lenguaje de programación para desarrollar productos para los sistemas operativos de Apple—, [Swift avanzado](/advanced-swift/intro), [UI Kit](/ui-kit/intro), [SwiftUI](/swiftui/intro), y Diseño de Aplicaciones.
